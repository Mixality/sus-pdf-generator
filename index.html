The System Usability Scale (SUS) PDF Generator is an open source web-based tool that allows for fast and easy creation of interactive PDF questionnaires for the System Usability Scale by John Brooke. It is part of the SUS Analysis Toolkit and includes 18 languages and versions of the SUS, even allowing custom variable descriptors (like product, system, app, website) or custom product names as the variable descriptor.
You can use the Default variable descriptor used in the language/variant selected, Bangor et al. (2008)'s suggestion: Product, the original suggestion System by Brooke (1996), or Custom to input a custom variable descriptor or your products name. Commonly used additional variable descriptors are: Website, Prototype, or App
